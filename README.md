# Description
A service that evaluates JavaScript code.

# How to run
- `docker-compose build`
- `docker-compose up`

# How to use
See the test: https://github.com/Hoobie/js-executor-service/blob/master/test/test.js

# Supported libraries
- Tesseract.js
- Tracking.js
- canvas
- atob
